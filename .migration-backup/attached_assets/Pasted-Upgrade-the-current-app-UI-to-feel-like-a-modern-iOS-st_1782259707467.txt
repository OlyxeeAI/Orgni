Upgrade the current app UI to feel like a modern iOS-style product.

Keep the existing functionality, routes, data flow, and core logic unchanged. Focus only on visual design, motion, spacing, interaction polish, and responsive behavior.

Design direction:

* Make the app feel clean, soft, premium, and mobile-first.
* Use an iOS-inspired visual system: rounded corners, soft shadows, subtle borders, blur effects, smooth transitions, and calm spacing.
* Use a bright clean background, preferably off-white or very light gray.
* Use glass-style cards where suitable, but do not overdo it.
* Use soft layered surfaces instead of flat blocks.
* Make buttons feel native and tactile with hover, active, and pressed states.
* Improve typography hierarchy with clear headings, readable body text, and balanced spacing.
* Use modern colors with one clear primary accent color.
* Avoid harsh colors, heavy shadows, sharp corners, clutter, or outdated dashboard styling.

Motion:

* Add smooth micro-interactions.
* Use gentle transitions for buttons, cards, navigation, modals, tabs, forms, and hover states.
* Add subtle entrance animations where appropriate.
* Avoid distracting or slow animations.
* Motion should feel like iOS: smooth, light, and responsive.

Components to improve:

* Navigation/sidebar/header
* Cards and panels
* Buttons
* Inputs/forms
* Tables/lists
* Empty states
* Modals/dialogs
* Loading states
* Mobile responsiveness
* Spacing and alignment

Technical requirements:

* Do not break existing functionality.
* Do not remove important content.
* Do not change API calls unless required for styling.
* Keep the code clean and reusable.
* Create or improve a shared design system if needed.
* Use consistent CSS variables or Tailwind theme values for colors, radius, spacing, shadows, and transitions.
* Make the UI work well on desktop and mobile.
* After changes, check for layout bugs, broken interactions, and unreadable text.

Goal:
The app should feel like a polished iOS-inspired SaaS/mobile web product: clean, smooth, modern, soft, and professional.
